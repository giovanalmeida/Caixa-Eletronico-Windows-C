# Caixa-Eletronico-Windows-C
Simulador de caixa eletrônico desenvolvido em C para Windows, com autenticação por PIN. Permite consulta de saldo, saques, depósitos e visualização de extrato de transações. Utiliza arquivos para armazenar dados e histórico de operações. Implementa validações como limites diários e controle de tentativas de acesso.
